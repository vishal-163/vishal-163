# About Me

I'm **Vishal R**, a Computer Science Engineering student at **K.S. Institute of Technology, Bangalore**, passionate about building modern web applications and AI-powered software.

I enjoy designing scalable full-stack applications with clean architecture, secure backend systems, responsive user interfaces and intelligent AI integrations. My primary focus is creating production-ready software that solves real-world problems while maintaining performance, scalability and reliability.

Alongside web development, I'm currently working on **WarTech Jacket**, a smart military defence wearable system for real-time soldier health monitoring and emergency response, and continuously exploring new technologies across AI, cloud computing and backend engineering.
